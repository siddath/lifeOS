# Evidence Log — artifacts, not ideas

> The core principle of this OS is *Think once → Build → Measure → Improve* — but "Measure" needs memory. Every shipped artifact gets one line here, newest on top. When the brain says "I've done nothing," this file is the counter-evidence.

**What counts as evidence:** a note, commit, issue, checklist, draft, decision, shipped feature, completed event, real conversation, sent message.
**What doesn't:** a plan, a reorganized plan, a better plan.
