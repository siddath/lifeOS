# 03_Projects — ventures and long-running builds

This folder holds your **long-running projects** — the ventures, side-builds, and multi-month efforts that aren't the single active mission but still need a home. A project here can be *promoted* into [`01_Focus/`](../01_Focus/) when it becomes the thing you're pushing, then dropped back to maintenance mode when the mission changes.

## What counts as a project

Anything with its own arc, its own artifacts, and a life longer than one week: a product you're shipping, a craft you sell, a business you're standing up, a course you're building. Not a task (that goes to [`../06_Trackers/tasks.md`](../06_Trackers/tasks.md)) and not a life area (that goes to [`../02_Areas/`](../02_Areas/)).

## Suggested layout

Give each project its own file or subfolder:

```
03_Projects/
  fieldnotes/          a side-project app
    README.md          what it is, current state, next action
    roadmap.md         phases and milestones
  pottery/             a craft venture
    README.md
```

## The one rule

**One project runs hot at a time.** The rest sit in *maintenance mode* — kept warm, next-action noted, but not competing for the mission slot. Running every project at full intensity at once is the spiral LifeOS is designed to prevent.

For a project you want to be able to talk about (interviews, pitches, retros), instantiate [`../templates/project_story.md`](../templates/project_story.md).

See [`../examples/alex/`](../examples/alex/) for how the demo persona runs "Fieldnotes" (hot) and "Weekend Pottery" (maintenance) side by side.
