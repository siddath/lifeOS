# worksheets/

Agent traces for long or risky tasks in **this repo's own development** — not your life data
(that lives in `06_Trackers/` and `dashboard/`).

One file per task, `YYYY-MM-DD-slug.md`, updated as the work goes so that if a session dies
partway, the next agent can resume from it. Useful shape: **Goal · Constraints · State
(newest first) · Next · Verification · Parking lot**. Commit the worksheet with the work.

Friction worth fixing goes in [`FEEDBACK.md`](FEEDBACK.md). The build queue is [`../TODOS.md`](../TODOS.md).
